# Chrome扩展

打开新标签页时获取知乎日报最新的一片文章。

## 安装

1. 克隆本仓库到本地

  ```sh
    git clone https://github.com/coderfe/newtab.git
  ```

2. 在Chrome浏览器中扩展程序中开启开发者模式，选择“加载已解压的扩展程序”。
